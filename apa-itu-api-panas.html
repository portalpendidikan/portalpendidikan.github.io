16. Jelaskan kenapa api panas!


Api bisa panas karena besar dari benda yang terbakar yaitu oksidasi berasal dari energi yang memiliki intensitas, dimana energi ini memiliki variasi sehingga selain panas juga dapat menghasilkan banyak warna sehingga dapat menimbulkan asap.

Bahan dasar api agar dapat terbentuk adalah udara, oksigen dan bahan material, dan jika salah satu bahan tersebut tidak ada maka api akan padam.

Tapi bisa panas intinya karena ada Oksigen yang melakukan persenyawaan dan reaksi dengan unsur-unsur lainnya.Api bisa panas karena adanya oksidasi oksigen dengan molekul udara di sekitarnya seperti karbondioksida, metana, dan lain-lain. Dalam ilmu termodinamika, oksidasi panas akan membuat udara sekitarnya pana sehingga kalor merambat secara radiasi karena tidak menggunakan zat perantara. Inilah yang menyebabkan api itu panas. Panasnya api akan terus apabila oksigen terus melakukan oksidasi sehingga menyebabkan panas yang merambat secara radiasi ke daerah di sekitarnya. Adanya panas ini akan terus meningkat karena kobaran api dan arah rambatan jangkauan radiasi panasnya meningkat sehingga api akan terus panas.

28. Kenapa api panas.....?


Karena mengandung mekulPanas merupakan salah satu jenis energi (energi kalor). Api dihasilkan dari reaksi pembakaran. Reaksi pembakaran merupakan reaksi antara bahan pembakar dengan oksidan. Reaksi ini terjadi pemutusan dan pengikatan atom. Untuk peristiwa pemutusan atom harus mempunyai energi yaitu panas. Pemutusan atom terjadi karena elektron-elektron di dalamnya mendapatkan panas untuk keluar dari atomnya. Setetelah terjadi pemutusan atom terjadi pembentukan atom baru dan api. Energi yang dilepaskan lebih besar daripada energi yang dibutuhkan. Api bersifat panas karena energi yang tersimpan pada reaksi pembakaran dilepaskan secara tiba-tiba.
