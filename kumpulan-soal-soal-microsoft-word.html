9. jelaskan cara membuka dokumen dengan nama file soal soal mid smester pada folder praktikum pada microsoft office word 2007


Ada 2 cara untuk membuka file tersebut
Cara 1 (lewat perintah Open di Word):
- Buka program Microsoft Office Word 
- Kemudian klik Office Button (untuk Word 2007) atau menu File (untuk Word 2010 keatas), kemudian klik ikon Open, atau bisa juga dengan menekan kombinasi tombol Ctrl+O pada keyboard 
- Lalu cari folder/tempat dimana file soal soal mid smester tersebut ada, pilih filenya, kemudian klik Open 

Cara 2 (lewat Explorer)
- Buka Explorer 
- Kemudian masuk ke folder dimana file soal soal mid smester tersebut berada 
- Kalau sudah ketemu filenya, klik 2x file tersebut, dan nanti akan muncul dokumen tersebut

2. Pilih tab "Insert" di menu atas.

3. Di grup "Illustrations" pada tab "Insert", pilih jenis gambar yang ingin Anda masukkan ke dokumen. Misalnya, Anda dapat memilih "Picture" untuk memasukkan gambar dari file komputer Anda atau "Shapes" untuk membuat gambar dengan menggunakan bentuk geometri.
