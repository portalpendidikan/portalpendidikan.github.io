Pakai aturan turunan untuk pembagian

Jika f(x) = u/v maka f'(x) = (u'v - uv')/v²

misal u = 3x - 3 maka u' = 3

misal v = 5x + 6 maka v' = 5

Masukkan ke rumus f'(x)

f'(x) = [ 3(5x + 6) - (3x - 3)5 ] / (5x + 6)²

f'(x) = (15x + 18 - 15x + 15) / (5x + 6)²

= 3 / (5x + 6)²

f'(x) = 3 / (25x² + 60x + 36)
