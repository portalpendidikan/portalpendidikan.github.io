3. Fungsi fitur title pada aplikasi windows movie maker adalah... *A. Teks pada clip videoB. Teks berjalan di akhir videoC. Jeda filmD. JudulE. Pemotongan durasi​


Fungsi fitur title pada aplikasi windows movie maker adalah Judul. Yang berari fiture ini digunakan untuk mengetahui judul film yang akan atau sedang ditonton.

Pembahasan :

     Windows Movie Maker (disingkat WMM) adalah  program pengeditan video sederhana yang dirancang untuk pemilik PC yang memiliki sedikit pengalaman dalam membuat video rumahan. Sebenarnya, Microsoft telah memperkenalkan fitur WMM ini beberapa waktu lalu di Windows Millennium Edition (Me). Namun, saat itu masih berupa suplemen atau aksesori. Dengan WMM ini Anda dapat melakukan pengeditan video sederhana dan memutarnya melalui Windows Media Player.  Anda kemudian dapat menyalinnya ke CD untuk membuat koleksi video.  

     Apa yang membuat Windows Movie Maker istimewa adalah ia bekerja dengan baik dengan camcorder digital dan analog.  WMM seharusnya bekerja langsung dengan camcorder DV. Namun, WMM ini juga mendukung kartu pengambilan PCI analog  dan webcam berbasis USB. Hal ini dapat dibuktikan dengan memasang camcorder DV. Oleh karena itu, jika Anda menjalankan Microsoft Windows XP, maka secara otomatis akan terdeteksi dan  Windows Movie Maker akan terbuka.

      Singkatnya, Windows Movie Maker adalah program atau perangkat lunak pengolah video yang berasal dari Windows XP. Movie Maker akan diinstal secara otomatis saat Anda menginstal Windows XP. Meskipun merupakan program built-in daripada program mandiri, Movie Maker bisa menjadi program pengeditan video  yang kuat. Hal ini tidak terlepas dari kemudahan penggunaan baik bagi pengguna pemula maupun pengguna tingkat lanjut, berkat dukungan Microsoft dalam memberikan update dan fitur tambahan pada website.

Menu Tambah memiliki fitur judul untuk menambahkan  slide khusus untuk judul video atau proyek.
