7. bagaimana cara membedakan tipe data integer dengan tipe data karakter?


Variabel merupakan salah satu komponen utama dalam sebuah program dimana informasi dinamis disimpan. Tidak semua variabel memiliki karakteristik yang sama. Karakteristik dari sebuah variabel sangat bergantung pada jenis atau tipe data variabel tersebut.


Terdapat berbagai jenis tipe data dalam setiap bahasa pemrograman. Salahs satunya adalah tipe data integer dan tipe data karakter. 


Cara membedakan tipe data integer dengan tipe data karakter adalah pada tipe data integer mengandung nilai numerik. Sedangkan tipe data karakter mengandung karakter ( termasuk juga huruf dan angka ). Tipe data karakter biasanya diiringi dengan penggunaan ", sebagai contoh " Ini adalah karakter". Sedangkan tipe data integer hanya berupa angka, contoh 123
