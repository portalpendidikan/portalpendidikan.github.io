1. Nilai maximum dari f(x)= x3 -12x +15 dalam interval -3 ≤ X. ≤ 1 adalah...


Nilai maksimum dari f(x) = x³ - 12x + 15 dalam interval -3 ≤ x ≤ 1 adalah...

Pembahasan :

-3 ≤ x ≤ 1
=> x = {-3, -2, -1, 0, 1}

f(x) = x³ - 12x + 15

f(-3) = (-3)³ - 12(-3) + 15
f(-3) = -27 + 36 + 15
f(-3) = 24

f(-2) = (-2)³ - 12(-2) + 15
f(-2) = -8 + 24 + 15
f(-2) = 31

f(-1) = (-1)³ - 12(-1) + 15
f(-1) = -1 + 12 + 15
f(-1) = 26

f(0) = 0³ - 12(0) + 15
f(0) = 0 - 0 + 15
f(0) = 15

f(1) = 1³ - 12(1) + 15
f(1) = 1 - 12 + 15
f(1) = 4

Jadi nilai maksimum dari f(x) adalah 31 di x = -2

CARA LAIN

dengan titik stasioner ==> f'(x) = 0
misal titik stasionernya di x = x₁
titik maksimum jika f''(x₁) < 0
titik minimum jika f"(x₁) > 0
titik belok jika f"(x₁) = 0

f(x) = x³ - 12x + 15
f'(x) = 3x² - 12

f'(x) = 0
3x² - 12 = 0
3(x² - 4) = 0
3(x + 2)(x - 2) = 0
x = -2 atau x = 2

f"(x) = 6x
f"(-2) = 6(-2) = -12 < 0 => maks di x = -2
f"(2) = 6(2) = 12 > 0 => minimum di x = 2

Jadi nilai maksimum dari f(x) adalah
f(-2) = (-2)³ - 12(-2) + 15
f(-2) = -8 + 24 + 15
f(-2) = 31

29. apa pengertian maximum advantage ?​


Jawaban:

Maximum advantage adalah istilah yang digunakan dalam teori perdagangan internasional yang menggambarkan keuntungan maksimal yang diperoleh oleh sebuah negara dalam memproduksi dan menjual produk tertentu di pasar internasional dengan menggunakan sumber daya yang dimilikinya.

Penjelasan:

Konsep ini menyatakan bahwa setiap negara dapat memperoleh keuntungan yang maksimal dalam produksi dan perdagangan dengan fokus pada produk-produk yang dapat diproduksi dengan biaya yang relatif lebih rendah daripada negara lain, dan mengimpor produk yang tidak dapat diproduksi secara efisien.

Dengan menerapkan konsep maximum advantage, suatu negara dapat meningkatkan kemakmuran ekonominya dengan mengoptimalkan penggunaan sumber daya yang dimilikinya, meningkatkan produksi dan ekspor produk-produk yang dapat diproduksi secara efisien, serta mengimpor produk-produk yang tidak dapat diproduksi secara efisien dan lebih murah dari negara lain. Hal ini diharapkan dapat memperkuat daya saing suatu negara di pasar internasional dan meningkatkan kesejahteraan masyarakatnya.

40. f(x) = -2x² + 4x + 7 tentukan nilai maximum​


Jawab:

Untuk mencari nilai maksimum dari sebuah fungsi kuadrat seperti F(x) = -2x² + 4x + 7, Anda dapat menggunakan rumus berikut: x = -b / (2a)

Dengan menggunakan rumus ini, kita dapat mencari nilai maksimum dari F(x) dengan menemukan nilai a, b, dan c dari fungsi tersebut. Untuk F(x) = -2x² + 4x + 7, nilai a, b, dan c adalah -2, 4, dan 7, masing-masing.

Jadi, nilai maksimum dari F(x) adalah: x = -b / (2a) = -4 / (-2 x 2) = 1.

Kemudian, Anda dapat menggunakan nilai x tersebut untuk mencari nilai F(x) pada titik maksimum, yaitu F(1) = -2 x 1² + 4 x 1 + 7 = 3. Jadi, nilai maksimum dari F(x) adalah 3.
