|3 + 2x| = |4 - x|

(3 + 2x)² = (4 - x)²
(3 + 2x)² - (4 - x)² =0
(3 + 2x + 4 - x) (3 + 2x - 4 + x) = 0
(x + 7) (3x - 1) = 0

x = -7

3x - 1 = 0
3x = 1
