8. pengertian inti sel beserta fungsi n gambarnya



Inti sel atau nukleus adalah organel yang ditemukan pada sel eukariotik. Organel ini mengandung sebagian besar materi genetiksel dengan bentuk molekul DNA linier panjang yang membentuk kromosombersama dengan beragam jenis protein. Gendi dalam kromosom-kromosom inilah yang membentuk genom inti sel.

Fungsi utama nukleus adalah untuk menjaga integritas gen-gen tersebut dan mengontrol aktivitas sel dengan mengelola ekspresi gen. Selain itu, nukleus juga berfungsi untuk mengorganisasikan gen saat terjadi pembelahan sel, memproduksi mRNA untuk mengkodekan protein, sebagai tempat sintesis ribosom, tempat terjadinya replikasi dan transkripsi dari DNA, serta mengatur kapan dan di mana ekspresi gen harus dimulai, dijalankan, dan diakhiri.

29. Diketahui suatu sudut, besar penyikunya 15 derajat lebih besar dari empat kali sudut tersebut. jika sudut tersebut adalah n derajat, tentukan besar n dan penyikunya? jelaskan?


Jawaban:

misal : sudut = n

penyiku = 90° - n

90° - n = 4n + 15°

90° - 15° = 4n + n

75° = 5n

n = 75° /5

n = 15°

penyiku = 90° - n

............. = 90° - 15°

............. = 75°

Penjelasan dengan langkah-langkah:

besar sudut = n

besar penyiku = 4n + 15°

sudut + penyiku = 90°

n + 4n + 15° = 90°

5n + 15° = 90°

5n = 90° - 15°

5n = 75°

n = 75° : 5

n = 15°

besar penyiku = 4n + 15°

= 4 x 15° + 15°

= 60° + 15°

= 75°

jadi besar sudutnya 15°, besar penyikunya 75°

35. Diberikan fungsi f(n)= n + 2, apakah f(n) merupakan fungsi ? (berikan penjelasannya) ​


Ya, fungsi f(n) = n + 2 adalah sebuah fungsi matematika.

Sebuah fungsi matematika adalah korespondensi yang menghubungkan setiap elemen dalam satu himpunan (domain) dengan tepat satu elemen dalam himpunan lainnya (codomain). Dalam hal ini, fungsi f(n) menghubungkan setiap bilangan n dengan hasil penjumlahan n + 2.

Pertama-tama, kita perlu memastikan bahwa setiap nilai n dalam domain memiliki hasil yang unik dalam codomain. Dalam fungsi f(n) = n + 2, ketika kita memberikan suatu nilai n, hasil penjumlahannya selalu unik. Misalnya, jika kita memberikan n = 1, maka f(1) = 1 + 2 = 3. Jika kita memberikan n = 2, maka f(2) = 2 + 2 = 4. Dalam setiap kasus, nilai n menghasilkan hasil yang berbeda dalam codomain.

Selanjutnya, fungsi f(n) memenuhi kriteria bahwa setiap elemen dalam domain memiliki korespondensi dengan tepat satu elemen dalam codomain. Dalam fungsi ini, setiap bilangan n dalam domain memiliki hasil penjumlahan n + 2 yang unik. Tidak ada dua bilangan n yang akan menghasilkan hasil yang sama. Misalnya, jika f(1) = 3, tidak ada nilai lain dalam domain yang akan menghasilkan 3 dalam codomain. Demikian pula, jika f(2) = 4, tidak ada nilai lain dalam domain yang akan menghasilkan 4 dalam codomain.

Dengan demikian, fungsi f(n) = n + 2 memenuhi kedua kriteria penting dalam definisi fungsi matematika, yaitu setiap nilai dalam domain memiliki hasil yang unik dalam codomain, dan setiap elemen dalam domain memiliki korespondensi dengan tepat satu elemen dalam codomain. Oleh karena itu, f(n) merupakan sebuah fungsi.
