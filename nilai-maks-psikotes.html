31. f (x)= 2 sin (x+pi/6) untuk 0 < x < 2pi tentukan nilai stasioner nilai maks dan min titik balik maks dan min fungsi naik turun


Jawaban:

Untuk mencari nilai stasioner, kita perlu mencari turunan pertama fungsi f(x) terlebih dahulu:

f(x) = 2sin(x + π/6)

f'(x) = 2cos(x + π/6)

Untuk mencari nilai stasioner, setel turunan pertama f(x) = 0:

2cos(x + π/6) = 0

cos(x + π/6) = 0

Kita mencari nilai-nilai x di antara 0 < x < 2π yang memenuhi persamaan ini. Nilai-nilai x yang memenuhi persamaan ini adalah:

x + π/6 = π/2, 3π/2, 5π/2, ...

x = π/2 - π/6, 3π/2 - π/6, 5π/2 - π/6, ...

x = π/3, 5π/6, 4π/3, ...

Jadi, titik-titik stasioner adalah x = π/3, 5π/6, 4π/3.

Untuk mencari nilai maksimum dan minimum, kita perlu memeriksa perubahan tanda dalam grafik fungsi. Karena fungsi ini adalah fungsi sinus dengan koefisien 2, nilai maksimumnya adalah 2 dan nilai minimumnya adalah -2.

Titik-titik balik maksimum terjadi ketika sinus memaksimumkan output sehingga sin(x + π/6) = 1. Titik-titik ini terjadi pada x = 0 + 2πn - π/6, dengan n adalah bilangan bulat.

Sehingga titik-titik balik maksimum adalah x = -π/6, 11π/6, 23π/6, ...

Titik-titik balik minimum terjadi ketika sinus meminimumkan output sehingga sin(x + π/6) = -1. Titik-titik ini terjadi pada x = π + 2πn - π/6, dengan n adalah bilangan bulat.

Sehingga titik-titik balik minimum adalah x = 5π/6, 17π/6, 29π/6, ...

Fungsi ini naik pada interval 0 < x < π/3 dan 4π/3 < x < 2π.

Fungsi ini turun pada interval π/3 < x < 5π/6, 5π/6 < x < 4π/3, dan pada interval -π/6 < x < 0.

Ini merupakan analisis mengenai nilai stasioner, nilai maksimum dan minimum, titik balik maksimum dan minimum, serta pola naik turun dalam fungsi f(x) = 2sin(x + π/6) untuk 0 < x < 2π.


32. nilai maks fungsi f(x) :30x-5x² adalah​


Fungsi f(x) adalah fungsi kuadrat berpola ax²+bx+c dengan a = -5, b = 30, dan c = 0

Nilai maksimum fungsi dapat dicari dengan rumus Yp = [tex]\frac{-D}{4a}[/tex]

Yp = [tex]\frac{-D}{4a} = \frac{-(b^{2}-4ac) }{4a} = \frac{-(30^{2}-4(-5)(0))}{4(-5)} = \frac{-900}{-20} = 45[/tex]

Jadi, nilai maksimum fungsi f(x) = 30x - 5x² adalah 45.
