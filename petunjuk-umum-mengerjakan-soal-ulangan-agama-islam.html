28. Dalam ulangan mtk, siswa mengerjakan 5 soal dengan 5 macam pilihajawaban setiap soalnya. Peluang siswa menjawab benar 3 soal adalah


Peluang siswa menjawab benar 3 soal adalah 3.

=> n(A) = 3
=> n(S) = 5
=> Peluang = P(A) = n(A) / n(S)
=> = 3/5 (peluang per soal) => Peluang menjawab benar 3 dari 5 soal
=> (3/5) × 5 = 3

Silahkan diteliti kembali jika ada kekeliruan
