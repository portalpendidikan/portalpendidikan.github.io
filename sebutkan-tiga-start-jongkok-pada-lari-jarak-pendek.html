1. Start pendek ( Bunch Start )

  Start pendek diawali dengan berdiri tegak dengan kedua kaki rapat. Kemudian, kedua tangan berada di samping badan. Adapun cara melakukannya adalah sebagai berikut:

a. Ambil ancang-ancang dengan posisi kaki kiri (kaki yang terkuat) diletakkan di depan dan ditekuk.

b. Posisikan lutut kaki kanan diletakkan di sebelah kaki kiri dan beri jarak sekitar satu kepal.

c. Posisikan kedua tangan diletakkan di belakang garis start dengan ibu jari terpisah dan empat jari lainnya merapat atau seperti membentuk huruf V.

d. Luruskan lengan kedua tangan dan fokuskan pandangan ke depan.

e. Angkat panggul hingga posisinya lebih tinggi dari pundak.

f. Lalu, tolak balok sekuat mungkin dengan kaki belakang. Mulailah berlari saat tanda sudah dimulai sudah dibunyikan.

2. Start menengah ( Medium Start )

    Seperti halnya start pendek, gerakan awal start menengah juga sama, yakni berdiri tegak lurus, kedua tangan berada di samping, dan fokuskan pandangan ke depan. Untuk cara melakukannya adalah sebagai berikut:

a. Ambil ancang-ancang posisi kaki kiri (kaki yang terkuat) diletakkan di depan dan ditekuk.

b. Posisikan lutut kaki kanan di sebelah kanan tumit kaki kiri dan beri jarak sekitar satu kepal lebih.

c. Kemudian, letakkan kedua tangan di belakang garis start dengan ibu jari terpisah dan empat jari lainnya merapat atau membentuk huruf V.

d. Luruskan lengan kedua tangan dan fokuskan pandangan ke depan.

e. Angkat panggul hingga posisinya lebih tinggi dari pundak.

f. Lalu tolak balok sekuat mungkin dengan kaki belakang. Mulailah berlari saat tanda sudah dimulai sudah dibunyikan.

3. Start panjang ( Long Start )

   Start panjang atau long start diawali dengan berdiri tegak, kedua tangan berada di samping badan, dan kaki dirapatkan. Cara melakukan start panjang adalah sebagai berikut:

a. Pertama, ambilah ancang-ancang dengan posisi kaki kiri (kaki yang terkuat) diletakkan di depan lutut kaki kanan dan ditekuk.

b. Terus posisikan lutut kaki kanan pada bagian belakang kaki kiri dan beri jarak sekitar satu kepal lebih.

c.  Kemudian, letakkan kedua tangan di belakang garis start dengan ibu jari terpisah dan empat jari lainnya merapat.

d. Luruskan lengan kedua tangan dan fokuskan pandangan ke depan.

e. Angkat panggul hingga posisinya lebih tinggi dari pundak.

f. Saat akan mulai berlari tolak balok sekuat mungkin dengan kaki belakang.

2. sebutkan tiga macam start jongkok dalm lari jarak pendek!​


Sebutkan macam-macam start jongkok!

Jawaban

Pendahuluan

Atletik dikenal sebagai induk dari berbagai cabang olahraga, yang merupakan gerak dasar manusia yaitu jalan, lari, lompat dan lempar. Cabang olahraga Atletik sendiri terbagi menjadi dua nomor yaitu nomor lintasan dan nomor lapangan. Dari nomor lintasan dan nomor lapangan itu terbagi lagi, untuk nomor lintasan terdiri dari nomor lari (Sprint) dan jalan (Run). Sedangkan di nomor lapangan terdiri dari nomor lompat (Jump) dan lempar (Throw).

Pembahasan

Dalam olahraga lari jarak pendek atau sprint, merupakan salah satu olahraga atletik cabang lari. Olahraga lari jarak pendek sendiri memiliki berbagai macam panjang lintasan, mulai dari 100, 200, hingga 400 meter.

Dalam olahraga lari jarak pendek, untuk startnya menggunakan start jongkok. Start Jongkok dalam olahraga lari pendek sendiri terbagi menjadi 3 macam, diantaranya ada Start Jongkok Posisi Pendek (Bunch Start), Start Jongkok Posisi Menengah (Medium Start), dan Start Jongkok Posisi Panjang (Long Start).

Kesimpulan

Ada 3 Macam Start Jongkok dalam Olahraga Lari Pendek, diantaranya :

Bunch Start (Start Pendek),

Medium Start (Start Menengah), dan

Long Start (Start Panjang).

8. sebutkan tahapan gerak start jongkok dalam lari jarak pendek


Sebagai seorang atletik dalam nomor lari, teknik start adalah salah satu teknik dasar yang wajib diketahui oleh setiap atlet. Teknik ini digunakan ketika hendak melakukan awalan saat akan berlari. Adapun teknik yang sering digunakan adalah start jongkok. Teknik start jongkok terdiri dari 3 macam, yakni start pendek, start menengah, dan start panjang.

Penjelasan:Cara melakukan start jongkok Posisi ‘bersedia’

Ketika panitia telah memberikan aba-aba ‘bersedia’, maka atlet harus segera berada di belakang garis start. Letakkan kedua tangan di belakang garis dengan jarak kedua tangan sedikit lebih lebar dari bahu. Bentuk huruf V dengan menggunakan jari. Letakkan salah satu kaki sekitar 40 cm di belakang garis start. Sejajarkan lutuk kaki yang di belakang dengan jari kaki yang ada di depan. Jarak lutut dan ujung jari kaki depan hanya sekitar satu kepalan tangan saja. Lemaskan kepala dan fokuskan pandangan ke depan.

Posisi ‘siap’

Selanjutnya ketika panitia telah memberikan aba-aba ‘siap’, angkat pinggul agak ke atas sehingga posisi bokong lebih tinggi daripada bahu. Pindahkan seluruh tumpuan berat badan ke tangan sehingga badan terasa ringan.Rendahkan posisi kepala, lalu pindahkan pandangan ke lintasan yang berjarak sekitar 1-1.5 meter dari garis start. Luruskan tangan untuk bersiap melakukan gerakan selanjutnya dan konsentrasi pada aba-aba berikutnya.

Posisi ‘ya’

Ketika panitia telah membunyikan tembakan atau memberikan aba-aba ‘ya’, maka segera angkat badan dan berlari sekuat mungkin dengan menolak badan menggunakan tumpuan kaki belakang. Condongkan badan ke depan untuk memaksimalkan kecepatan saat mulai berlari.

Sprint atau lari cepat/lari jarak pendek adalah  jenis lari yang dilakukan di sepanjang garis lintasan dengan kekuatan dan kecepatan penuh dari awal hingga akhir, pemenang ditentukan berdasarkan catatan waktu tersingkat. Nomor punggung sprint adalah nomor punggung 100m, 200m dan 400m. Macam-macam start jongkok antara lain: Short start, medium start, dan long start. Cara melakukan start jongkok adalah: tempatkan lutut kaki belakang kita di ujung kaki depan kita dengan kepalan tangan, rentangkan lengan lurus sejajar dengan bahu dan buat bentuk V terbalik dengan telapak tangan, pandangan lurus ke trek, bobot tubuh berada di kedua tangan, dengan sinyal bahwa bobot siap untuk bergerak maju, dengan sinyal untuk menembak ke depan dengan cepat. Lomba lari jarak menengah ini terbuka untuk pelari pria atau wanita. Biasanya jaraknya adalah 800, 1500  dan 3000 meter. Lari jarak menengah sedikit berbeda dengan lari sprint.

Atletik adalah olahraga tertua di dunia. Atletik merupakan induk atau pendahulu dari berbagai cabang olahraga seperti lempar, lompat, dan lari. Atletik sering diadakan di berbagai kejuaraan dunia seperti olimpiade. Ada banyak jenis olahraga atletik.

Lompat jauh.  Lompat jauh adalah gerakan melompati kotak lompat sambil mempertahankan pusat gravitasi di udara selama mungkin. Lompat jauh adalah cabang olahraga lompat dalam cabang olahraga atletik. Lari jarak pendek. Sprint adalah perlombaan lari  jarak 100m, 200m, dan 400m. Sprint membutuhkan kecepatan untuk memenangkan pertandingan. Estafet ke-3. Perlombaan estafet adalah olahraga atau perlombaan lari yang dijalankan oleh tim. Setiap tim sprint terdiri dari 4  pemain. Ekiden berlari dengan  tongkat yang diteruskan ke rekan setimnya ke garis finis. Lempar tombak. Lempar lembing adalah olahraga lempar yang merupakan bagian dari trek dan lapangan. Lempar lembing melibatkan melempar tombak menggunakan teknik tertentu dan melemparkannya sejauh mungkin.

27. Tuliskan macam-macam start jongkok pada lari jarak pendek​


Start jongkok adalah start yang dilakukan dengan posisi badan jongkok di belakang garis start. Start jongkok terdiri dari tiga macam, yakni start pendek, start menengah dan start panjang.

Pembahasan

Start jongkok merupakan teknik start yang digunakan dalam beberapa olahraga atletik, salah satunya adalah lari jarak pendek. Start jongkok dilakukan dengan posisi badan jongkok, dengan salah satu lutut diletakkan di tanah. Start jongkok terdiri dari tiga macam, yakni.

Start pendek. Start pendek dilakukan dengan posisi kedua telapak kaki berada di papan pijakan dan kedua tangan menyentuh tanah di belakang garis start. Posisi lutut kaki depan ditekuk dan diletakkan di depan dada. Sedangkan posisi lutut kaki belakang diletakkan di tanah dan berada di depan jari-jari kaki depan.  Start menengah. Start menengah dilakukan dengan posisi kedua telapak kaki berada di papan pijakan dan kedua tangan menyentuh tanah di belakang garis start. Posisi lutut kaki depan ditekuk dan diletakkan di depan dada. Sedangkan posisi lutut kaki belakang diletakkan di tanah dan berada sejajar dengan tumit kaki depan.  Start panjang. Start pendek dilakukan dengan posisi kedua telapak kaki berada di papan pijakan dan kedua tangan menyentuh tanah di belakang garis start. Posisi lutut kaki depan ditekuk dan diletakkan di depan dada. Sedangkan posisi lutut kaki belakang diletakkan di tanah dan berada 1 jengkal di belakang tumit kaki depan.

Jawaban:

3 Macam Teknik Start Jongkok: Pendek, Menengah, dan Panjang

- Teknik start jongkok merujuk pada posisi permulaan pelari ketika mengawali lomba yang menempatkan diri dengan posisi jongkok (crouching) di garis mula (start).

Seperti dikutip dari buku Kepelatihan Atletik Jalan dan Lari (2018) karya Suratmin, teknik awalan crouching start atau start jongkok umumnya digunakan dalam perlombaan lari jarak pendek.

Start jongkok terdiri atas tiga macam, yaitu start jongkok pendek, menengah, dan panjang.

Menguasai teknik awalan atau start dalam olahraga lari, khususnya lari jarak pendek, bisa menentukan prestasi seorang pelari dalam perlombaan.

Perbedaan yang mencolok antara lari jarak pendek dengan jarak menengah terletak pada posisi start karena lari jarak menengah secara umum menggunakan awalan berdiri (standing start).
