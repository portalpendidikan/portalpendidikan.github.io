13. jelaskan cara menggunakan stack dalam informatika!!​


Jawaban:

Stack merupakan salah satu struktur data yang digunakan untuk menyimpan sekumpulan data. Sama halnya seperti konsep sebuah tumpukan dimana yang berada paling atas atau akhir akan diambil terlebih dahulu, begitupun stack pada informatika bersifat LIFO (last in first out).

Stack memiliki beberapa operasi yang dapat digunakan seperti di bawah.

1. Pop, operasi ini digunakan untuk menghapus elemen pada stack.

2. Push, operasi ini digunakan untuk memasukan elemen pada stack.

3. Peek, operasi ini digunakan untuk mengecek data teratas dari stack.

4. isFull dan isEmpty yang digunakan untuk mengecek apakah stack penuh atau kosong.


14. 1. jelaskan pengertian komunikasi menurut asal katanya!2. uraikan tentang pencarian beruntun dengan sentinel!.3. terangkan proses pada bubble sort!.4. sebutkan fungsi dan kegunaan dari struktur data stack!.5. sebutkan langkah-langkah untuk melakukan operasi dequeue!. bantu jawab kak"ಠ⁠﹏⁠ಠ​


Jawaban:

Baik, berikut adalah jawabannya:

1. **Komunikasi menurut asal katanya adalah proses berbagi makna dengan orang lain.** Kata "komunikasi" berasal dari bahasa Latin "communis", yang berarti "bersama". Komunikasi dapat dilakukan dengan berbagai cara, seperti berbicara, menulis, membaca, dan menyimak.

2. **Pencarian beruntun dengan sentinel adalah algoritme pencarian yang menggunakan nilai sentinel untuk mempermudah proses pencarian.** Nilai sentinel adalah nilai yang tidak mungkin ada dalam daftar data. Algoritme pencarian beruntun dengan sentinel dimulai dengan membandingkan nilai sentinel dengan elemen pertama dalam daftar data. Jika nilai sentinel lebih besar dari elemen pertama, maka elemen pertama tidak diproses. Jika nilai sentinel lebih kecil dari elemen pertama, maka elemen pertama diproses. Proses ini diulang untuk elemen-elemen berikutnya dalam daftar data hingga elemen yang dicari ditemukan.

3. **Bubble sort adalah algoritme pengurutan yang bekerja dengan cara membandingkan elemen-elemen yang berdekatan dan menukarnya jika elemen yang kiri lebih besar dari elemen yang kanan.** Proses ini diulang hingga elemen-elemen dalam daftar terurut.

4. **Fungsi dan kegunaan dari struktur data stack adalah untuk menyimpan data secara terbalik.** Stack dapat digunakan untuk menyimpan data yang akan diproses secara berurutan, seperti data dalam antrian atau data dalam riwayat browser.

5. **Langkah-langkah untuk melakukan operasi dequeue adalah sebagai berikut:**

* Hapus elemen pertama dari stack.

* Kembalikan elemen yang telah dihapus.

18. Susunlah sebuah program C++ untuk menyiapkan array 1 dimensi yang digunakan sebagai Stack S sebanyak 5 elemen, bertipe integer. Kemudian lakukan proses simpan data ke stack (PUSH) atau proses mengeluarkan isi stack (POP) dengan proses sebagai berikut:1. Inputkan data dari keyboard. Bila data yang diinputkan bernilai 999, maka proses selesai.2. Bila data yang diinput bernilai >= 60, maka periksa kondisi stack. Bila stack masih bisa diisi, maka simpan data tersebut (PUSH) kedalam stack, dan proses diulang kembali mulai no 1. Tapi bila stack sudah penuh, data tidak jadi disimpan, kemudian cetak perkataan “Stack Penuh”, dan proses selesai.3. Bila data yang diinputkan <60, maka periksa kondisi stack. Bila stack ada isinya, maka ambil isi stack dan cetak ke layar, kemudian proses diulang kembali mulai no. 1. Bila stack taka da isinya, maka cetak perkataan “ Stack Kosong”, dan proses selesai. Kemudian:​


Program Stack untuk C++ adalah sebagai berikut yaitu:

#include <iostream>

using namespace std;

int stack[999], n=999, top=-1;

void push(int val) {

  if(top>=n-1)

  cout<<"Stack Overflow"<<endl;

  else {

     top++;

     stack[top]=val;

  }

}

void pop() {

  if(top<=-1)

  cout<<"Stack Underflow"<<endl;

  else {

     cout<<"The popped element is "<< stack[top] <<endl;

     top--;

  }

}

void display() {

  if(top>=0) {

     cout<<"Stack elements are:";

     for(int i=top; i>=0; i--)

     cout<<stack[i]<<" ";

     cout<<endl;

  } else

  cout<<"Stack is empty";

}

Penjelasan:

Tumpukan adalah  struktur data yang terbentuk dari urutan unit data yang terbatas. Di tumpukan, Anda dapat menambahkan dan menghapus elemen hanya dari  satu tempat di bagian bawah tumpukan. Posisi ini disebut top atau top of the stack. Stack digunakan untuk mengontrol operasi  sistem operasi. Selain itu, stack juga merupakan algoritme hebat yang dapat Anda gunakan untuk membuat phaser (membaca serangkaian operasi dari sebuah rumus). Dalam ilmu komputer, tumpukan adalah kumpulan objek yang bekerja sesuai dengan prinsip LIFO (last in first out). H. Data yang terakhir  dimasukkan meninggalkan tumpukan terlebih dahulu.
