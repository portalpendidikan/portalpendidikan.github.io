= 2^5 x 2^3

Jawab: x = 5, y = 5

= 2^5 x 2^5

= 32 x 32

= 1024 koin

Tinggi tumpukan = tebal koin x jumlah koin

= 0,2 x 1024

= 204,8 cm

10. jika a+b=1, b+c=2, dan c+a=3, maka a+b+c=tolong jawab ya soalnya bimbel kelas VII susah​


Jawaban

a + b + c = 3

Penjelasan dengan Langkah-langkah

diketahui :

a + b = 1 kemungkinan a = 1 dan b = 0.

b + c = 2 kemungkinan c = 2

c + a = 3 bisa dipastikan a = 1 dan c = 2

jadi

a = 1 , b = 0 , c = 2

jawab :

a + b + c

1 + 0 + 2 = 3

~Bilangan bulat

a + b = 1

b + c = 2

c + a = 3

a = 1 - 0 = 1

b = 0 - 0 = 0

c = 3 - 1 = 2

Maka :

a + b + c

= 1 + 0 + 2

= 3✔

•••

●Kesimpulan :

a + b + c = 3✔
