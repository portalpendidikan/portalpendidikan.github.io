Buat 2 soal PG berikut kuncinya terkait indikator "Siswa dapat menentukan nama barisan, jika diberikan suatu barisan bilangan"Buat 2 soal PG berikut kuncinya terkait indikator "Siswa dapat menentukan suku ke-n barisan aritmetika, jika diketahui suku-suku yang lain#Jawab Sesuai Dengan Indikator​​


Jawaban:

A). Barisan Aritmatika dan Geometri

Barisan aritmatika ialah barisan yang berpasangan dengan barisan geometri.

Kalau bedanya yaitu memiliki rumus yang beda :

Barisan aritmatika memiliki rumus :

Un = a + (n - 1)b → untuk menentukan Suku ke-n

Sn = n/2 (2a + (n - 1)b → untuk menentukan jumlah suku ke-n

Kalau barisan geometri :

Un = arⁿ-¹ → untuk menentukan suku ke-n

Sn = a (rⁿ - 1) / (r - 1) → untuk menentukan jumlah suku ke-n

keterangan :

a = suku pertama , b = beda suku

Un = suku ke-n , Sn = jumlah suku ke-n

Contoh soal Pilihan Ganda :

Barisan Aritmatika seperti 3, 5, 7, 9, ...dan seterusnya

memiliki beda pada setiap barisannya.

Barisan Geometri seperti 4 + 8 + 16 + 32 + .. + dan seterusnya

memiliki rasionya pada setiap barisannya.

1). Diketahui barisan aritmatika 4,6,8,10,..., 52. carilah suku ke 20 barisan tersebut ?

a. 40

b. 41

c. 42

d. 44

Jawaban : (C)

Penjelasan :

a = suku pertama = 4

b = U² – a = 6 - 4 = 2

Un = a + (n - 1)b

U20 = 4 + (20 - 1)2

U20 = 4 + (19)2

U20 = 4 + 38

U20 = 42

2). Barisan 3 + 6 + 12 + 24,....,+ 48 . Membentuk barisan geometri , tentukan suku ke 8 barisan tersebut.

a.384

b.388

c. 432

d. 300

Jawaban : (A)

penjelasan :

a = suku pertama = 3

r = rasionya = 6/3 = 2

Un = arⁿ-¹

U8 = 3.2^(8-1)

U8 = 3.2^7

U8 = 3 × 128

U8 = 384

B). Menentukan Rumus suku ke-n

Barisan Aritmatika

Contoh soal PG nya :

1). Tentukan rumus suku ke-n pada barisan 5, 10, 15, 20, dan seterusnya.

a. Un = 5n + 1

b. Un = 5n

c. Un = 1 - 5n

d. Un = 4n

Jawaban : (B)

Penjelasan :

a = suku pertama = 5

b = beda sukunya = 10 - 5 = 5

Maka, rumus suku ke-n

Un = a + (n - 1)b

Un = 5 + (n - 1)5

Un = 5 + 5n - 5

Un = 5n - 0

Un = 5n

2). Barisan Geometri

Carilah rumus suku ke-n pada barisan berikut 2 + 4 + 8 + 16 + 32 + ... dan seterusnya.

a. Un = 2ⁿ+¹

b. Un = 2ⁿ-¹

c.Un = 2ⁿ

d.Un = 4ⁿ

Jawaban : (C)

Penjelasan :

a = suku pertama = 2

r = rasionya = 4/2 = 2

maka, rumus suku ke -n

Un = arⁿ-¹

Un = 2. 2ⁿ-¹

Un = 2¹. 2ⁿ-¹

Un = 2^(1 + n - 1)

Un = 2ⁿ
