3. jika Ip 192.168.18.177 brp ip subnet dan ip broad cast


Jika IP tersebut memiliki Mask 255.255.255.0 / Class C Full
maka IP Subnet adalah 192.168.18.0 dan IP Broadcast 192.168.18.255Cara menghitung IP Subnet (IP Network):
Ubah IP menjadi biner:
11000000.10101000.00010010.10110001
Tidak ada suffix (Nilai CIDR)? Gunakan default netmask: /24 yaitu 255.255.255.0.
Ubah Subnet mask tsb menjadi biner juga:
11111111.11111111.11111111.00000000

Perhitungan Network:
Subnet mask AND IP:
[tex]11111111.11111111.11111111.00000000[/tex]
[tex]11000000.10101000.00010010.10110001[/tex]
-------------------------------------------------------- AND
[tex]11000000.10101000.00010010.00000000[/tex] ATAU 192.168.18.0

Perhitungan Broadcast:
Invers Subnet Mask OR Network
[tex]00000000.00000000.00000000.11111111[/tex]
[tex]11000000.10101000.00010010.00000000[/tex]
------------------------------------------------------------------- OR
[tex]11000000.10101000.00010010.11111111[/tex] ATAU 192.168.18.255

Maka dengan IP 192.168.18.177, 
subnet masknya adalah 255.255.255.0

Networknya adalah 192.168.18.0 dengan broadcast 192.168.18.255

7. Hal atau faktor apa yang menjadi pemisah dari 2 bagian alamat IP yaitu network dan host ? Subnet Mask Broadcast Address Default Gateway DNS Server​


Jawaban:

Hal atau faktor yang menjadi pemisah dari dua bagian alamat IP, yaitu network dan host, adalah:

1. Subnet Mask: Subnet mask digunakan untuk memisahkan bagian network dan host dalam alamat IP. Subnet mask adalah sebuah bilangan biner atau desimal yang digunakan untuk menentukan sejauh mana bagian alamat IP yang digunakan untuk mengidentifikasi jaringan (network) dan bagian yang digunakan untuk mengidentifikasi perangkat (host). Subnet mask memberikan petunjuk tentang bagian mana dari alamat IP yang merupakan bagian network dan bagian host.

2. Broadcast Address: Broadcast address adalah alamat IP khusus yang digunakan untuk mengirim pesan broadcast ke semua perangkat dalam suatu jaringan. Dalam jaringan yang menggunakan subnetting, broadcast address digunakan untuk mengirim pesan broadcast dalam subnet tertentu. Broadcast address adalah alamat IP dengan bagian host yang seluruh bitnya diatur ke nilai biner 1.

3. Default Gateway: Default gateway adalah alamat IP perangkat jaringan yang digunakan sebagai pintu gerbang (gateway) untuk menghubungkan satu jaringan dengan jaringan lainnya. Default gateway biasanya merupakan alamat IP perangkat router yang menghubungkan jaringan lokal dengan jaringan eksternal atau internet. Default gateway memungkinkan perangkat dalam jaringan untuk mengirimkan paket data ke tujuan di luar jaringan lokal.

4. DNS Server: DNS server (Domain Name System server) adalah server yang bertanggung jawab untuk menerjemahkan nama domain menjadi alamat IP yang sesuai. DNS server membantu dalam proses resolusi nama domain menjadi alamat IP yang diperlukan untuk mengarahkan koneksi ke tujuan yang diinginkan.

Semua faktor di atas berperan dalam memisahkan dan mengarahkan bagian network dan host dalam alamat IP serta dalam pengiriman data antara jaringan dan ke tujuan yang diinginkan.

6. Untuk menghitung berapa banyak subnet dan host per subnet, kita perlu melakukan beberapa langkah:

Hitung jumlah bit untuk subnetting dengan menghitung berapa banyak bit "1" pada subnet mask. Dalam hal ini, jumlah bit "1" pada subnet mask adalah 26.Tentukan jumlah bit untuk subnetting dengan mengurangi jumlah bit pada alamat IP yaitu 32 bit dengan jumlah bit untuk subnetting (26). Dalam hal ini, jumlah bit untuk subnetting adalah 6.Tentukan jumlah subnet dengan menghitung 2 pangkat jumlah bit untuk subnetting. Dalam hal ini, jumlah subnet adalah 64.Tentukan jumlah host per subnet dengan mengurangi jumlah bit pada alamat IP yaitu 32 bit dengan jumlah bit untuk subnetting (26) dan dengan menambah satu. Dalam hal ini, jumlah host per subnet adalah 62.
