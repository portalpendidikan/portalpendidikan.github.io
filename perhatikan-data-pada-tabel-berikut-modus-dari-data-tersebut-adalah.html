25. perhatikan data pada tabel berikut nilai 60,65,70,75,80,85,90,95 banyak siswa 1,4,2,10,11,3,1,1 tentukan raata rata,median,dan modus dari data di atas​


Rata - rata data tersebut adalah 74,24.Median data tersebut adalah 75.Modus data tersebut adalah 80.

Nilai rata - rata (dilambangkan dengan x̄) pada suatu kelompok adalah suatu bilangan yang mewakili keseluruhan data pada kelompok tersebut yang didapatkan dari jumlah nilai data dibagi dengan banyaknya data tersebut. Biasanya nilai rata - rata digunakan untuk mengevaluasi data atau membandingkan kelompok data yang satu dengan yang lainnya sehingga lebih cepat dan menggambarkan seluruh data namun tidak dapat digunakan untuk menentukan nilai data tertentu di antara sekelompok data.

Rata - rata suatu data ditentukan dengan rumus :

x̄ = [tex] \frac{jumlah \: data}{banyaknya \: data} [/tex]

Median adalah nilai tengah data. Untuk menentukan median suatu data, pertama data yang masih acak harus diurutkan dari yang terkecil hingga terbesar, kemudian data yang telah diurutkan tersebut ditentukan nilai tengahnya. Apabila banyak data ganjil maka nilai median langsung dapat diperoleh. Adapun jika banyak data genap maka nilai median sama dengan jumlah kedua data di tengah dibagi dua.

Sedangkan modus adalah data yang paling sering muncul atau terjadi. Untuk memudahkan menentukan modus, pertama susun data dalam urutan meningkat atau sebaliknya, kemudian hitung frekuensinya. Modus digunakan baik untuk tipe data numerik atau pun data kategoris karena tidak dipengaruhi oleh nilai ekstrem.

Agar lebih jelas dalam penerapannya, simak pembahasan soal berikut ini.

PEMBAHASAN :

Diketahui suatu data memuat nilai siswa sebagai berikut :

1 orang siswa mendapat nilai 60,

4 orang siswa mendapat nilai 65,

2 orang siswa mendapat nilai 70,

10 orang siswa mendapat nilai 75,

11 orang siswa mendapat nilai 80,

3 orang siswa mendapat nilai 85,

1 orang siswa mendapat nilai 90, dan

1 orang siswa mendapat nilai 95.

Tentukan nilai rata - rata, median dan modusnya.

Dari data di atas kita dapat melihat bahwa jumlah siswanya adalah : 1 + 4 + 2 + 10 + 11 + 3 + 1 + 1 = 33 orang. Maka,

RATA - RATA

Rata - rata data di atas adalah :

x̄ = [tex] \frac{jumlah \: data}{banyaknya \: data} [/tex]

x̄ = [tex] \frac{(1 \times 60) + (4 \times 65) + (2 \times 70) + (10 \times 75) + (11 \times 80) + (3 \times 85) + (1 \times 90) + (1 \times 95)}{33} [/tex]

x̄ = [tex] \frac{60 + 260 + 140 + 750 + 800 + 255 + 90 + 95}{33}[/tex]

x̄ = [tex] \frac{2.450}{33} [/tex]

x̄ = 74,24

MEDIAN

Karena datanya ganjil (n = 33), maka data mediannya terletak pada : X[tex]( \frac{n + 1}{2})[/tex]

= X[tex]( \frac{33 + 1}{2})[/tex]

= X17 atau data ke - 17

Sementara letak data ke - 17 adalah nilai 75, maka median data tersebut adalah 75.

MODUS

Dari data di atas, nilai yang memiliki frekuensi paling banyak adalah 80, maka modus data tersebut adalah 80.
