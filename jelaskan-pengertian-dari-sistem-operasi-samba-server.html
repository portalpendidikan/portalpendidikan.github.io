21. Bagaimana Samba Server dapat digunakan sebagai layanan pencetakan (print service)? Jelaskan  tentang k onfigurasi printer pada Samba Server, termasuk cara menginstal driver printer pada Samba  Server dan mengatur hak akses untuk pengguna dan grup yang dapat menggunakan printer tersebut.​


Jawaban:

Konfigurasi Printer pada Samba Server termasuk persiapan, penginstalan driver (jika diperlukan), dan menentukan hak akses untuk pengguna dan grup.

Persiapan:

- Pastikan driver printer yang tepat sudah terinstal untuk sistem operasi yang digunakan.

- Pastikan Samba Server telah terinstal.

- Geser printer ke server Samba.

Penginstalan Driver:

- Buka halaman web yang menyediakan driver printer untuk sistem operasi yang Anda gunakan.

- Unduh driver printer dan simpan ke folder yang mudah Anda temukan dengan CMD.

- Masukkan kata sandi root jika diminta untuk melakukannya.

- Setelah seluruh unduhan selesai, cabut (atau turn-off) printer dari port.

- Sambungkan kembali printer ke port.

- Setelah printer terhubung, lanjutkan dengan menjalankan proses instalasi driver.

Pengaturan Hak Akses:

- Buka “Manajemen Pencetakan” atau “Kelompok Pencetakan” melalui halaman “Layanan dan Aplikasi”.

- Klik kanan pada printer yang baru saja didukung dan klik atribut printer.

- Di dialog “Hak Akses”, tentukan siapa yang boleh menggunakan printer dengan memilih opsi simpan.

- Tentukan juga siapa yang tidak boleh menggunakan printer dengan memilih opsi “Diblokir”.

- Akhiri dengan mengklik “Done”.

Konfigurasi printer telah selesai. Sekarang printer yang terpasang pada Samba Server akan siap digunakan.

28. Jelaskan sistem operasi sebagai gatekeeper, accountant, dan server


Sistem operasi adalah perangkat lunak yang menjadi jembatan antara pengguna dan mesin sebuah perangkat.

Pembahasan

Pada kesempatan ini, soal meminta kita untuk menjelaskan fungsi sistem operasi sebagai gatekeper, accountant, dan server. Berikut kakak akan mencoba menjawab pertanyaan tersebut.

Fungsi sistem operasi

Sebagai gatekeper, sistem operasi berfungsi mengendalikan seluruh akses ke dalam dan luar sistem termasuk aktivitas di dalam sistem.Sebagai accountant, sistem operasi berfungsi mengendalikan kinerja perangkat termasuk pemakaian CPU, pemakaian memori, pemanggilan fungsi tertentu, penyimpanan data, serta koneksi antarbagian.Sebagai server, sistem operasi berfungsi menyajikan layanan yang dibutuhkan oleh kalangan pengguna.

40. jelaskan yg di maksud dengan sistem operasi server


Server adalah sebuah sistem komputer yang menyediakan jenis layanan tertentu dalam sebuah jaringan komputer. Server didukung dengan prosesor yang bersifat scalable dan RAM yang besar, juga dilengkapi dengan sistem operasi khusus, yang disebut sebagai sistem operasi jaringan atau network operating system. Server juga menjalankan perangkat lunak administratif yang mengontrol akses terhadap jaringan dan sumber daya yang terdapat di dalamnya, seperti halnya berkas atau alat pencetak (printer), dan memberikan akses kepada workstation anggota jaringan. 

Umumnya, di atas sistem operasi server terdapat aplikasi-aplikasi yang menggunakan arsitektur klien/server. Contoh dari aplikasi ini adalah DHCP Server, Mail Server, HTTP Server, FTP Server, DNS Server dan lain sebagainya. Setiap sistem operasi server umumnya membundel layanan-layanan tersebut atau layanan tersebut juga dapat diperoleh dari pihak ketiga. Setiap layanan tersebut akan merespons terhadap request dari klien.
