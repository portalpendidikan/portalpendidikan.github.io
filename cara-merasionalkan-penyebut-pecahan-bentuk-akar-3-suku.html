20. bentuk pecahan dengan penyebut bilangan rasional dari pecahan -2 per 3 - akar 3​


Bentuk pecahan dengan penyebut bilangan rasional dari pecahan -2 per 3 - akar 3​ adalah [tex]-\frac{3+\sqrt{3}}{3}}[/tex]

Pembahasan

Apabila pecahan mengandung penyebut berupa bentuk akar maka pecahan tersebut harus dirasionalkan penyebutnya dengan cara :

a) mengalikan dengan bentuk akar yang sama

[tex]\frac{a}{\sqrt{b}}=\frac{a}{\sqrt{b}}\times\frac{\sqrt{b}}{\sqrt{b}}=\frac{a\sqrt{b}}{b}[/tex]

b) mengalikan pembilang dan penyebut dengan sekawan dari penyebutnya. Misal penyebutnya adalah (a - √b) maka sekawannya (a +√b)

[tex]\frac{a}{a-\sqrt{b}}=\frac{a}{a-\sqrt{b}}\times\frac{a+\sqrt{b}}{a+\sqrt{b}}[/tex]

=========================================================

[tex]-\frac{2}{3-\sqrt{3}}=-\frac{2}{3-\sqrt{3}}\times{-\frac{3+\sqrt{3}}{3+\sqrt{3}}}[/tex]

[tex]-\frac{2}{3-\sqrt{3}}=-\frac{2(3+\sqrt{3})}{3^2-(\sqrt{3})^2}[/tex]

[tex]-\frac{2}{3-\sqrt{3}}=-\frac{2(3+\sqrt{3})}{9-3}[/tex]

[tex]-\frac{2}{3-\sqrt{3}}=-\frac{2(3+\sqrt{3})}{6}[/tex]

[tex]\boxed{-\frac{2}{3-\sqrt{3}}=-\frac{3+\sqrt{3}}{3}}[/tex]

Menyederhanakan bentuk akar:

Cara menyederhanakan bentuk akar yaitu dengan membagi bilangan tersebut dengan bilangan pangkat sesuai akarnya, contoh:

[tex] \sqrt{50} = \sqrt{2 \times 25} = \sqrt{2} \times \sqrt{25} = 5 \sqrt{2} [/tex]

↑ Merupakan akar pangkat 2, bilangan pangkatnya adalah 25, setelah itu dijadikan bilangan desimal yaitu 5 karena 5²=25.

Contoh lagi:

[tex] \sqrt[3]{81} = \sqrt[3]{3 \times 27} = \sqrt[3]{3} \times \sqrt[3]{27} = 3 \sqrt[3]{3} [/tex]

↑ Merupakan akar pangkat 3, bilangan pangkatnya adalah 27, setelah itu dijadikan bilangan desimal yaitu 3 karena 3³=27.

Merasionalkan penyebut pecahan bentuk akar

[tex] \frac{a}{ \sqrt{b} } = \frac{a}{ \sqrt{b} } \times \frac{ \sqrt{b} }{ \sqrt{b} } [/tex]

Contoh:

[tex] \frac{2}{ \sqrt{5} } = \frac{2}{ \sqrt{5} } \times \frac{ \sqrt{5} }{ \sqrt{5} } = \frac{2 \sqrt{5} }{5} [/tex]
