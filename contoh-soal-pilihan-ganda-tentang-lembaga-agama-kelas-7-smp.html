37. tolong pilihan ganda kelas 7 SMP pakai cara​


Jawaban:

B. 130°

Penjelasan dengan langkah-langkah:

Sudut segitiga keseluruhan = 180°

180° = 100° +30° + B

180° = 130° + B

B = 50°

ditanyakan x

x = 180° - 50°

X = 130°

Penjelasan dengan langkah-langkah:

Jumlah ketiga sudut dalam segitiga adalah 180°, maka :

a + 30° + 100° = 180°

a + 130° = 180°

a = 180° - 130°

a = 50°

Sudut yang terdapat pada garis lurus adalah 180° juga, maka :

a + x = 180°

50° + x = 180°

x = 180° - 50°

x = 130° (B.)

Jadi, besar sudut x adalah 130° (B.)
